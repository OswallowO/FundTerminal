FEATURES

為認真試算而生

不是隨便畫條線,而是用券商級資料口徑,把報酬算到對。

📈

含息還原的真實報酬

ETF 採含息還原收盤價、基金採累積淨值,並自動偵測除權息 / 分割造成的價格斷崖回補,模擬不失真。

🗓️

定期定額模擬

自選每月扣款日(可複選)、扣款金額與起始本金,逐日算出單位數、市值、報酬率與年化 XIRR。

🧺

投資組合回測

多檔標的各自獨立設定(不同買進日、本金、扣款),合併成組合總走勢,附每檔投入 / 市值 / 報酬 / 權重。

⚖️

同現金流對照大盤

把你的每一筆投入「改買加權指數」算出對照報酬,apples-to-apples 比較你贏大盤多少。

📑

專業報表匯出

PDF 報告(績效摘要、持倉明細、走勢圖、每日明細)、Excel 多工作表與 CSV,一鍵帶走。

🔄

桌面終端 · 自動更新

深色專業介面、互動式走勢圖(線性 / 對數切換),單一執行檔免安裝;有新版主畫面一鍵更新。

三步開始

不用安裝、不用註冊,下載就能跑。

下載並執行

到 Releases 下載 FundTerminal.exe,雙擊執行 —— 免安裝,程式會自動開啟瀏覽器顯示介面。

設定標的與條件

搜尋基金 / ETF(如 0050、瀚亞),選起始日、填本金或定期定額金額與扣款日。

模擬與匯出

按下開始模擬看 KPI 與走勢;切到投資組合分頁可多檔合併;結果一鍵匯出 PDF / Excel / CSV。
