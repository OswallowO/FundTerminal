## 這是什麼

把「如果我從某一天開始,定期定額或單筆投入某檔基金 / ETF,到今天會變多少?」這個問題,用**真實歷史淨值**精準算給你看,並和**同期間的加權指數**做公平對照。支援把多檔標的組成**投資組合**一起回測。

純粹是歷史數據的試算工具,**不是投資建議**。

---

## ✨ 功能特色

- **含息還原的真實報酬** — ETF 採 Yahoo Finance 還原收盤價(含息),基金採累積淨值;自動偵測除權息 / 分割造成的價格斷崖並回補,模擬不失真。
- **定期定額模擬** — 自選每月扣款日(可複選)、扣款金額、起始本金,逐日計算單位數、市值、報酬率與年化(XIRR)。
- **投資組合回測** — 多檔標的各自獨立設定(不同買進日、本金、扣款),合併成組合總走勢;附每檔的投入 / 市值 / 報酬 / 權重明細。
- **同現金流對照大盤** — 把你的每一筆投入「改買加權指數」算出對照報酬,apples-to-apples 比較你贏大盤多少。
- **專業報表匯出** — PDF 報告(績效摘要、持倉明細、走勢圖、每日明細)、Excel 多工作表、CSV。
- **桌面終端體驗** — 深色專業介面、互動式走勢圖(可切線性 / 對數刻度)、單一執行檔免安裝。
- **自動更新** — 有新版時主畫面會提示,一鍵自動下載、替換、重啟。

## ⬇️ 下載與安裝

1. 前往 **[Releases 頁面](https://github.com/OswallowO/FundTerminal/releases/latest)**,下載最新版的 `FundTerminal.exe`。
2. 直接**雙擊執行**即可 —— 免安裝、不寫登錄檔;程式會自動開啟你的瀏覽器顯示介面。
3. 首次執行 Windows SmartScreen 可能跳出提示(未簽章的個人軟體常見),點「**其他資訊 → 仍要執行**」即可。

> 程式會在 exe 旁邊建立一個 `data/` 資料夾存放下載的歷史資料快取,可安全刪除。

---

## 🖥️ 使用方式

1. 在左側「基金名稱」輸入關鍵字(例:`高科技`、`元大`、`0050`)搜尋並選擇標的。
2. 選起始日期、填本金或定期定額金額與扣款日,按 **開始模擬**。
3. 想比較多檔?切到 **投資組合** 分頁,逐一「加入持倉」後按 **模擬組合**。
4. 結果可一鍵匯出 PDF / Excel / CSV。
